Projects

NYC Property Insights Report: Crime, Rent, and Airbnb Modeling

A comprehensive analysis combining exploratory data analysis (EDA) of NYC crime, rental income, and Airbnb listings to uncover key trends in the property rental market. The project involves data merging, feature engineering, sentiment analysis and extensive preprocessing to prepare the datasets for predictive modeling. Through strategic model selection, logistic regression is used to predict Superhost status, while linear regression estimates booking volumes. This analysis provides actionable insights into how crime rates, rental prices, and neighborhood dynamics impact Airbnb success in New York City.
